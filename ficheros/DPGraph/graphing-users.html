Who can download from this page: You can download the full version of DPGraph from this page if you are an employee, staff member, faculty member, or student, at any of the listed institutions. You may use DPGraph on any of your computers at home, work or school. If you are not covered by a legacy site license, you can buy an individual copy or you can download the free DPGraph Viewer.


Linux installation: DPGraph runs with the aid of a free program called Wine. Wine translates Windows system calls into Linux system calls, allowing many Windows programs to run under Linux. Wine is a work in progress, but DPGraph runs fairly nicely under recent versions.

  1. If your Linux distribution doesn't already come with Wine, then download and install Wine from www.winehq.com.
  2. Click on the name of your institution below to download a file called InstallDPGraph.exe (about 400k).
  3. From inside your window manager, issue the command wine InstallDPGraph.exe to install DPGraph.
  4. Issue the command wine "c:\program files\dpgraph\dpgraph.exe" to run DPGraph.

Please let us know if you run into abnormalities; there are many versions of Linux out there. Performance notes: 1) setting graph3d.box := false may increase performance on Linux systems that are slow at drawing text; 2) using the framebuffer device in 16M color modes seems to increase performance on many Linux systems.

Windows installation: The Windows version runs under Windows 95/98/00/ME/NT4/XP/Vista/7/8/10.

  1. Click on the name of your institution below to download a file called InstallDPGraph.exe (about 400k). It will be sent to your download directory (for many users, the download directory is the Desktop).
  2. Run InstallDPGraph.exe. For most users, that means click on Open when the download is complete, or click on the new InstallDPGraph.exe icon on the Desktop.

If you can't figure out where Windows sent InstallDPGraph.exe, click on Start, then click on Find, then search for InstallDPGraph.exe.
