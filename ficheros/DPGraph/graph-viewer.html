Click here to download the free DPGraph Viewer
Installation instructions are below.

DPGraph Viewer runs under Windows 95/98/00/ME/NT4/XP/Vista/7/8/10, or under Linux for 386 or later PC's. DPGraph Viewer will also run under Windows emulators such as SoftWindows or Virtual PC for the Mac.

DPGraph Viewer allows you to view and manipulate 2D through 8D graphs created by the full version of DPGraph. DPGraph Viewer comes with hundreds of examples contributed by users from around the world. DPGraph Viewer will also work with your web browser to enable you to view and manipulate graphs that are posted on the web (for example, see the DPGraph Gallery or documentation).

You may freely distribute the installation file for DPGraph Viewer. For example, if you are writing a textbook you can put a copy of InstallDPGraphViewer.exe on any CD's or diskettes that accompany your textbook (or, you can point your readers to http://www.dpgraph.com/graph-viewer.html so they can be sure to get the most recent version). If you are developing web-based content, you can copy InstallDPGraphViewer.exe to your own server if you would like.

Linux installation: DPGraph Viewer runs with the aid of a free program called Wine. Wine translates Windows system calls into Linux system calls, allowing many Windows programs to run under Linux. Wine is a work in progress, but DPGraph Viewer runs fairly nicely under recent versions.

  1. If your Linux distribution doesn't already come with Wine, then download and install Wine from www.winehq.com.
  2. Click on the link above to download a file called InstallDPGraphViewer.exe (about 380k).
  3. From inside your window manager, issue the command wine InstallDPGraphViewer.exe to install DPGraph Viewer.
  4. Issue the command wine "c:\program files\dpgraph viewer\dpgraphv.exe" to run DPGraph Viewer.

Please let us know if you run into abnormalities; there are many versions of Linux out there. Performance notes: 1) the less text on the graphs, the faster they will draw on many Linux systems; 2) using the framebuffer device in 16M color modes seems to increase performance on many Linux systems.

Windows installation: The Windows version runs under Windows 95/98/00/ME/NT4/XP/7/8/10.

  1. Click on the link above to download a file called InstallDPGraphViewer.exe (about 330k). It will be sent to your download directory (for many users, the download directory is the Desktop).
  2. Run InstallDPGraphViewer.exe. For most users, that means click on Open when the download is complete, or click on the new InstallDPGraphViewer.exe icon on the Desktop.

If you can't figure out where Windows sent InstallDPGraphViewer.exe, click on Start, then click on Find, then search for InstallDPGraphViewer.exe.
